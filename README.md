# PCA_TALOS - Archaeological NLP Concept Analysis

## Project Overview

This repository contains R code for performing Principal Component Analysis (PCA) on Natural Language Processing (NLP) concepts extracted from over 8,000 archaeological journals. The project was developed as part of a bibliometric analysis to identify key thematic patterns and concept distributions in archaeological literature.

## Description

The analysis processes concept frequency data from archaeological publications, applying dimensionality reduction techniques to uncover underlying patterns in the textual corpus. The workflow includes:

- **Data Import**: Loading concept frequency matrices from CSV files
- **Exploratory Analysis**: Initial data inspection and summary statistics
- **Statistical Analysis**: Implementation of PCA for dimensionality reduction
- **Visualization**: Generation of PCA plots to identify concept clusters and patterns

## Dataset

The analysis uses concept count data derived from:
- **Source**: 8,000+ archaeological journals
- **Method**: Natural Language Processing (NLP) techniques
- **Content**: Concept frequency matrices and bibliometric indicators
